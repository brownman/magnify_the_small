Introduction

Although, there are a lot of tutorials already on D3, I wanted to show examples in coffeescript with the added benefit of being able to change the code directly in the browswer. Hopefully, this will allow you to experiment with the examples to get a better understanding of how D3 works. My focus is on basic business type visualizations such as bar charts, line charts and pie charts.

A Basic Bar Chart

This first example shows all the basic elements needed to display output in SVG format

Successfully Compiled
 chart = d3.select("#canvas-barch
 
 numbers = [4, 88, 15, 16, 23, 12,4, 8, 15, 16, 23, 12,4, 8, 15, 13, 4, 5]
 chartHeight = 298
 
 chart = d3.select("#canvas-barchart1")
  .append("svg")
     .attr("width", "100%")
     .attr("height", "100%");
 
 chart.selectAll("rect").data(numbers).enter()
 .append("rect")
   .attr("y", (d, i) -> chartHeight - (d * 10))
   .attr("x", (d, i) -> (i * 20))
   .attr("width", 20)
   .attr("height", (d) -> d * 10);
